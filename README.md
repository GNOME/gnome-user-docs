gnome-user-docs
===============

This project contains the user and system administration guides for the
GNOME project.

You can browse the documentation online here: <https://help.gnome.org>

## Installation

You will need the following tools to build the GNOME user documentation:

- [meson](https://mesonbuild.com)
- [yelp-tools](https://gitlab.gnome.org/GNOME/yelp-tools/)
- [xmllint](https://gnome.pages.gitlab.gnome.org/libxml2/xmllint.html)

## Contributing

Please, follow the [contribution guide](./CONTRIBUTING.md) to know how to
contribute to the GNOME user documentation.

## Reporting Problems

If you find problems in the documentation, please report bugs:

https://gitlab.gnome.org/GNOME/gnome-user-docs/issues/

## Discussion

If you have questions related to the user documentation, you can ask on
[GNOME's Discourse instance](https://discourse.gnome.org/tag/documentation).
Alternatively, you can join [the Docs room on
Matrix](https://matrix.to/#/#docs:gnome.org).
